# student-management-system-c
student management system in c using File Handling implementing CRUD operations.
This project is  a console-based management system developed using the c programming language and binary file handling.
## FEATURES
-Add student
-Search student
-Update student
-Delete student
-count total student
## Concepts used
-structures
-File handling
-CRUD operations
-Menu Driven Programming
## How it works
-The system stores student records in a binary file('student.dat')
user can perform operations like adding, updating, searching,deleting, and counting students through menu driven interface.
